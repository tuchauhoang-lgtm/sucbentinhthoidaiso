# Tin nhắn gửi kèm source

Source riêng cho trang workshop `/laodong`.

Bạn không cần dùng Firebase/domain của mình. Bạn hãy tự tạo Firebase project riêng, đổi nội dung workshop, đổi link Google Form đăng ký, rồi deploy theo hướng dẫn trong file `README_HUONG_DAN.md`.

Nếu bạn không rành kỹ thuật, phần quan trọng nhất là:

- Sửa nội dung trong `suc_ben_hud/lao_dong/index.html`.
- Thay link `https://forms.gle/your-registration-form` bằng link Google Form của bạn.
- Nhờ người biết Firebase giúp chạy lệnh deploy.
